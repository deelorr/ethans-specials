# Ethan's Specials Library — V2

This is a mobile-first static web app.

## Open locally
Unzip and open `index.html` in a browser.

## Put it on your phone
Upload these files to GitHub Pages, Netlify, or Vercel. Then open the URL on your iPhone and use Share → Add to Home Screen.

## Files
- `index.html` — app shell
- `styles.css` — mobile-first styling
- `app.js` — filters, detail cards, week builder
- `data.js` — your embedded specials database
- `manifest.webmanifest` — basic install support
